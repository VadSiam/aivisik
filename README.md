# AIVisik

## About function which generating AI image by random country ggl trends and post it to insta

## Installation

```bash
$ npm install
```

add .env file - check .example_env file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
